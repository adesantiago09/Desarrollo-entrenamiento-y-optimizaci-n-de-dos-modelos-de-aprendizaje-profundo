# Datasets utilizados

Los datasets **no se incluyen** de forma local para mantener el repositorio ligero.

Se descargan automáticamente desde `keras.datasets` al ejecutar los notebooks:

- **CIFAR-10** → imágenes 32×32 de 10 clases para clasificación con CNN
- **IMDB Reviews** → reseñas de películas para análisis de sentimiento con RNN

> No es necesario descargar nada manualmente.
