# Desarrollo-entrenamiento-y-optimizaci-n-de-dos-modelos-de-aprendizaje-profundo
Proyecto de la materia Redes Neuronales y Sistemas Difusos: desarrollo, entrenamiento y optimización de dos modelos de aprendizaje profundo (CNN y RNN) aplicados a clasificación de imágenes (CIFAR-10) y análisis de sentimiento (IMDB), con evaluación y análisis de resultados.
